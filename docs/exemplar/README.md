# Exemplar: Wave Survival Arena

This folder shows what your project should look like at each step. Use it as a reference — your game will be different, but the structure should be similar.

## Game: Wave Survival Arena (2-Player Co-op)
Two players survive waves of enemies by shooting together. Enemies get faster and more numerous with each wave.

## Team
- Alex — players.js (both player characters, movement, lives)
- Jordan — enemies.js (enemy spawning, wave system, AI patterns)
- Sam — bullets.js + collisions.js + ui.js (shooting, all collisions, menus, HUD)

- `step4-component-map.md` — What the component plan looks like
